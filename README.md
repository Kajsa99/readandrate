# readandrate

Book review app where you can rate and comment on books.

## Tech
- Frontend: React + TypeScript + Vite
- Backend: Express + MySQL

## Setup

Start MySQL:
```bash
docker start readandrate-mysql
```

Backend:
```bash
cd backend
npm install
npm run dev
```

Frontend:
```bash
cd frontend
npm install
npm run dev
```

Frontend runs on http://localhost:5173
Backend runs on http://localhost:3000
